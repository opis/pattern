### Installation

This library is available on [Packagist](https://packagist.org/packages/opis/pattern) and can be installed using [Composer](http://getcomposer.org).

```bash
composer require opis/pattern
```

```json
{
    "require": {
        "opis/pattern": "^1.0.0"
    }
}
```
